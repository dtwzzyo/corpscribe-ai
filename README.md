# CorpScribe AI

**CorpScribe AI** это прототип локальной и безопасной RAG-системы (Retrieval-Augmented Generation) для ответов на вопросы по внутренней базе знаний.

Этот проект был создан для демонстрации навыков в построении full-stack ML-систем, от обработки данных до развертывания с помощью Docker.

##  Технический стек

*   **LLM Сервер:** Ollama (локальный инференс модели Mistral 7B)
*   **Бэкенд:** Python 3.9
*   **Оркестрация RAG:** LangChain
*   **Векторная база:** ChromaDB
*   **API Сервер:** Flask
*   **Веб-интерфейс:** Streamlit
*   **Контейнеризация:** Docker

---

##  Архитектура

Проект построен на клиент-серверной архитектуре:

1.  **LLM-сервер (ПК с GPU):** На нем запущен `Ollama`, который предоставляет LLM-модель через API.
2.  **Приложение (Docker-контейнер):**
    *   **Flask API (`app.py`):** Принимает запросы от UI, обрабатывает их с помощью RAG-цепочки LangChain и обращается к LLM-серверу.
    *   **Streamlit UI (`ui.py`):** Предоставляет пользователю простой и интуитивный чат-интерфейс.

---

##  Ключевые возможности

*   **Точные ответы:** Система отвечает строго на основе предоставленных документов, минимизируя "галлюцинации".
*   **Цитирование источников:** Каждый ответ сопровождается ссылками на фрагменты исходных документов.
*   **API-first дизайн:** Логика RAG обернута в REST API, что позволяет легко интегрировать ее с любыми другими сервисами.
*   **Полная контейнеризация:** Все приложение упаковано в Docker-контейнер для легкого запуска и масштабирования.

## Как запустить

**Предварительные требования:**
*   Docker и Docker Compose
*   Ollama, запущенная на ПК с GPU в той же локальной сети.

**1. Клонируйте репозиторий:**
```bash
git clone https://github.com/dtwzzyo/corpscribe-ai
cd corpscribe-ai
```

**2. Настройте LLM-сервер:**
*   Убедитесь, что ваш `Ollama` сервер доступен по сети.
*   Создайте файл `app.py` и `ingest.py` и впишите IP-адрес вашего ПК в переменную `PC_IP_ADDRESS`.

**3. Создайте папку `docs` и положите в нее ваши `.txt` файлы.**

**4. Соберите и запустите Docker-контейнер:**
```bash
docker build -t corpscribe-ai .
docker run -p 8501:8501 -p 5001:5001 corpscribe-ai
```

**5. Откройте приложение в браузере:**
Перейдите по адресу `http://localhost:8501`.
